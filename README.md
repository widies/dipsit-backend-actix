# How to start this project on local

## First clone this repository
`git clone https://github.com/widies/dipsit-backend-actix.git`

## Install the required dependencies using cargo
`cargo install`

## Setting Up Env file
Setting the variable environment on .env file. You can copy and rename the .env.sample into .env file.

## Running project
`cargo run`

## Running project for development
`cargo watch -x run`
